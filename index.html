BCI / Python decoder / actuator safety

BCI Prosthetic Leg Control

Public robotics/BCI portfolio project for translating EEG motor-imagery features into safe actuator trigger commands for a prosthetic-leg test rig.

  • Built live OpenBCI/Cyton decoding with BrainFlow, filtering, CSP feature extraction, and LDA scoring.
  • Added dry-run-first safety gates, thresholding, refractory timing, and REST-trial output blocking.
  • Published replay reports that separate in-session signal quality from cross-session generalization risk.
